# hyper-new-tab
Add a button `+` to easily open a new tab in your hyper terminal.

* Compatible with any theme
* Simple installation

<div align="center">
  <p>
    <img alt="demo" src="demo/default.png" width="600px" />
  </p>
</div>

### Install

`hyper i hyper-new-tab`

### Usage

Click `+` and a new tab will open.
